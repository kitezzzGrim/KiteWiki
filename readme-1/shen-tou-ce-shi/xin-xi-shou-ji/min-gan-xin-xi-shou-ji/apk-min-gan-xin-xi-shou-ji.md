# APK敏感信息收集

工具地址：[https://github.com/TheKingOfDuck/ApkAnalyser](https://github.com/TheKingOfDuck/ApkAnalyser)

## ApkAnalyser

一键提取安卓应用中可能存在的敏感信息。

目前提取了APK内:

* 所有字符串
* 所有URLs
* 所有ip
* 可能是hash值的字符串
* 存在的敏感词（如oss.aliyun）
* 可能是accessKey的值
